# Expense Tracker

## 📌 About
This is an Expense Tracker built using Python and Streamlit.

## ✨ Features
- Add expenses
- View expenses
- Delete expenses
- Total spendings
- Store data using CSV
- Expense visualization using pie chart

## 🛠️ Technologies Used
- Python
- Streamlit
- CSV
- Pandas

## 🚀 How to Run

```bash
streamlit run app.py
```
